параметры CSS 
caption {
    caption-side: top/bottom           заголовок до или после таблицы
    text-align: left/right/center      слева/справа или по центру
table {                            
    border: 3px solid black;     
    border-collapse: collapse;
    border-spacing: 3px;
    cellspacing: 3px;
    background-color                   цвет фона. 16ричное RGB-значение/16 базовых цветов
    color                              цвет текста
    border-color                       цвет границ
    width: auto/100%/333px             ширина таблицы
	height:auto/100%/111px             высота таблицы
	text-align: left/center/right.     выравнивание по горизонтали
    vertical-align: top/middle/bottom  выравнивание по вертикали
    }
th{}    
td {
    border: 3px solid lightgray;       толщина границ ячеек
    border-top/right/bottom/left:...   граница с указанной стороной
    padding: 3px;                      отступ от края до содержимого
    padding-top/right/bottom/left:...  
    }

теги таблиц 
 <table>    собственно таблица
 <caption>  заглавие таблицы
	<thead>    заглавная строка
 <tr>       строка
 <tr>       ячейка как заглавие
 <td>       обычная ячейка
 <colgroup> группы столбцов таблицы
 <col>      значения атрибутов для одного/нескольких столбцов таблицы
 <thead>    заголовок группы строк таблицы
 <tbody>    тело группы строк таблицы
 <tfoot>    нижний заголовок группы строк таблицы
	атрибуты table
      border="1"                  ширина внешней рамки(в пикселах)
      align="left/center/right"   горизонтальное выравнивание. deafult = left
      valign="top/bottom/middle"  вертикальное выравнивание
      cellpadding="1"             расстояние между рамкой ячейки и содержимым.
      cellspacing="1"             расстояние между ячейками.
      width="1"                   ширина таблицы (в пикселах/в процентах к ширине окна браузера). -"-.
      height="1"                  высота таблицы. -"-. deafult - от объема содержимого.
      bgcolor="green/..."         цвет фона. шестнадцатиричное RGB-значение/16 базовых цветов
      background="url/name.png"   фон таблицы рисунком
	атрибуты th & td
      colspan="3"    объединение ячеек по горизонтали. NB! количество ячеек   
      rowspan="2"    объединение ячеек по вертикали    NB! с учетом объединений
